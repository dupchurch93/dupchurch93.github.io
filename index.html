Welcome

My name is Dan and I'm a software engineer.

Howdy, my name is Daniel Upchurch. I am full stack software engineer living in Baltimore, Maryland.

I've always had a creative mind, but was sadly born with no rythm or artistic ability at all. I leaned towards creative problem solving instead and eventually studied physics due to my curious nature. After graduating I discovered another outlet for my creativity, programming! I quickly fell in love with coding due to the numerous types of problems and more importantly, the many out of the box approaches one can take to solving them. I immersed myself in a coding bootcamp, App Academy, and refined my skills in JavaScript, Python, and other technologies while building multiple full stack web applications. When I'm not seeking out new challenges I also enjoy playing ultimate frisbee or reading any fantasy/sci-fi books I can get my hands on.

My Portfolio

Martial Sheets

Live Link Github

Martial Sheets is a character creation tool for Dungeons and Dragons 5th Edition that I built using React and Flask. It streamlines the experience of creating and leveling up martial classes by dynamically showing what options are available and has an explanation for all the stats along the way. Styling was all done with TailwindCSS and I created the character sheet myself, modeling after the official D&D 5E sheet with a few changes.

Community Table

Live Link Github

Designed with Texas in mind after the power grid failure due to record breaking weather, Community Table is a resource sharing React application with a Flask backend. It utilizes the Google Maps API to show users any resources around them that are available and allows users to claim the amount they need, along with posting anything they may wish to share as well. Along with a group of 3 other developers I worked on all aspects of the project, however my main contribution was implementing all features with Google Maps.

GoodCooks

Live Link Github

Another group project I worked on using Express.js on the backend and PUG templates on the frontend, GoodCooks is a GoodReads clone but for recipes. Users can save recipes, mark them as cooked, or rate and review them. I worked closely with another dev to implement the AJAX requests for users to save recipes to their cupboards and add ratings to recipes.
